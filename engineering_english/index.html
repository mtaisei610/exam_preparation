lisp製の情報工学英語演習の練習ツールです。

How to use

1. Download all files.

2. Install SBCL.

3. Run quiz.lisp script.

# 単語練習 $ sbcl --script quiz.lisp y1.lisp # 文練習 $ sbcl --script quiz.lisp y2.lisp
